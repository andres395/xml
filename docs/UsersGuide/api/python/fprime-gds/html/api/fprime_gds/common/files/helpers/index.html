fprime_gds.common.files.helpersÂ¶

helpers.py:

Helpers for file uplink, and downlink. This contains shared components, classes, and support architecture in order to enable both file uplink and downlink to share the same structures. This includes the following shared objects:

  1. Timeout: for managing timeouts and timeout responses

  2. FileStates: managing the state of an uplink or downlinking file

  3. CFDPChecksum: calculates the CFDP checksum for files

  4. TransmitFile: file object for up and down
