fprime_gds.common.utils.config_managerÂ¶

@brief Utility class to read config files and provide configuration values

After the first instance of this class is initialized, any class can obtain the same instance by calling the static getInstance function. This allows any part of the python program retrieve configuration data.

Based on the ConfigManager class written by Len Reder in the fprime Gse
