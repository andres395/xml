@brief Base class for system data classes.

This class defines the interface for cata classes which are intended to hold a specific data item (packet, channel, event). This data item includes the time of the data as well as data such as channel value or argument value.

@date Created July 2, 2018 @author R. Joseph Paetz (rpaetz@jpl.nasa.gov)

@bug No known bugs
