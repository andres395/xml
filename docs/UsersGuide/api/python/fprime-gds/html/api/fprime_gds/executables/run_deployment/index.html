fprime_gds.executables.run_deployment.launch_process(cmd, logfile=None, name=None, env=None, launch_time=5)[source]¶

Launch a child subprocess. This subprocess will allow the child to run outside of the memory context of Python.

Parameters

  • cmd – list of command arguments to run by handing to subprocess.

  • logfile – (optional) place to redirect output to for purposes of logging. Default: None, screen.

  • name – (optional) short name for printing messages.

  • env – (optional) environment to run in. Allows for special environment contexts.

  • launch_time – (optional) time to launch the process, before rendering an error.

Returns

running process
