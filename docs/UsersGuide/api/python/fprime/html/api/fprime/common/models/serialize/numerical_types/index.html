fprime.common.models.serialize.numerical_types¶

numerical_types.py:

A file that contains the definitions for all the integer types provided as part of F prime. F prime supports integers that map to stdint.h integer sizes, that is, 8-bit, 16-bit, 32-bit, and 64-bit signed and unsigned integers.

@author mstarch
